

This repository provides a Python-based framework that combines first-principles electronic structure methods with advanced quantum many-body techniques.

Key features include:

Integration of Density Functional Theory (DFT) with Dynamical Mean-Field Theory (DMFT)
Use of a DFT-derived Hamiltonian as input for many-body calculations
Computation of:
Single-particle Green’s functions
Two-particle Green’s functions
Solution of the quantum impurity problem using the Continuous-Time Quantum Monte Carlo (CT-QMC) solver from the ALPS Project library

This framework is designed for studying strongly correlated electron systems with realistic material-specific inputs.
